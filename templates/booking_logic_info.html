Automatic Conversion Rules
When Does Auto-Conversion Happen?
  1. Service Booking: Contact books any non-consultation service
  2. Appointment Confirmation: Status changes to "confirmed" or "scheduled"
  3. Manual Admin Action: Admin clicks "Convert to Client" button
Calendar Booking Logic
  • Consultations: Remain as contacts (inquiry stage)
  • Service Appointments: Auto-convert to clients
  • Manual Bookings: Admin can choose conversion
  • Returning Customers: Always book as existing clients
